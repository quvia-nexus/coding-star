논리 연산자는 두 개 이상의 조건을 한꺼번에 판단할 수 있는 도구예요.


예를 들어 이렇게 생각해볼 수 있어요:

  • 🌧️ "비가 오지 않고"
  • ☀️ "날씨가 따뜻하면"
  • → 나갈 수 있다!

이처럼 "그리고(and)", "또는(or)", "아니다(not)"라는 말을 코딩에서도 사용할 수 있어요!

📝 문제

아래 코드의 ??? 에 알맞은 논리 연산자를 넣어 결과를 맞춰보세요.

📥 입력 예시

condition1 = (5 > 10) # False
condition2 = (7 == 7) # True
result = condition1 ??? condition2
print(result) # True

📤 출력 예시

True
